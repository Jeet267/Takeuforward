# Premium Wall Calendar 🗓️

An interactive, responsive, and aesthetically premium Wall Calendar component built with React. This project focuses on high-end design principles, including glassmorphism, dynamic animations, and month-specific themes.

## ✨ Features
- **Dynamic Month Themes**: Every month has a unique color palette and curated high-quality hero image based on the season.
- **Advanced Date Range Selection**: Select a start and end date with smooth hover previews and range highlights.
- **Persistent Notes**: Integrated note-taking for each month, persisted locally via `localStorage`.
- **Responsive Design**: Fully optimized for mobile (stacked view) and desktop (side-by-side view) layouts.
- **Premium Animations**: Built with `framer-motion` for smooth month transitions and state changes.

## 🎨 Design Choices
- **Typography**: Uses 'Outfit', a geometric sans-serif font for a modern and clean aesthetic.
- **Glassmorphism**: Subtle transparency and backdrop blurs are used in the notes and grid panels to create depth.
- **Interactive States**: Micro-interactions on date cells and navigation buttons enhance the tactile feel.
- **Visual Branding**: A slanted, dynamic-color overlay in the header mimics the look of a physical artistic wall calendar.

## 🛠️ Tech Stack
- **React 19**
- **Vite** (for fast development)
- **Framer Motion** (for premium animations)
- **Lucide React** (for vector icons)
- **Vanilla CSS Modules** (for scoped, high-performance styling)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Jeet267/Takeuforward.git
   cd Takeuforward
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
